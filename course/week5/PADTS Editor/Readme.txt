*************************** Description *******************************

PADTS Editor is an application that create effects in 8 bits BMP images
such as Negative, Brighter, Gray scale, Red Scale, etc.

*************************** Complements *******************************

The complements that the app needs are:

	*Tkinter  
	*ImageMagik

*************************** Instruccions ******************************

1. Run the script Complements.sh to install the complements.
2. Run the script GraficInterface.py to run the app.

************************* Operation Mode ******************************

1. Click on "Start" to remove unnecessary files.
2. Click on "Convert" to edit the images.
3. Click on whatever effect button to show the edited image.
4. Click on "Original" to show the original image.

** After you clicked on "Convert" you have to press the key "Enter" on 
the linux terminal if you don't press the key you couldn't watch the 
images.
