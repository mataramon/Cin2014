TextEditor
   Text editor with basic functions.

== Team ==
   Raul Calzada - Israel Sanchez

== Description ==
   The realization of this project was with Qt Creator software and several of its libraries.
   Compiled for the Windows operating system by the MinGW software.
   For the installer, NSIS program was used.

== Installation ==
   Run the installer and select the folder where you want to install.
